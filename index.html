    CSS Selector Crash Course

    The asterisk (*) aka universal selector for CSS, matches all elements in a document, allowing you to apply styles universally.

    It can be useful for resetting styles or applying a default style across all elements, but should be used carefully to avoid performance issues.


    Theres 3 types of CSS:

    1. In-Line: Applied directly within an HTML element's style attribute, affecting only that specific element.
    2. Internal: Defined within a style tag in the head section of an HTML document, influencing the entire page.
    3. External: Contained in a separate .css file linked to the HTML document, allowing for consistent styling across multiple pages.
